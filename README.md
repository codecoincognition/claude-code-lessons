# Claude Code Lessons

**Go from zero to dangerous with Claude Code — one hands-on lesson at a time.**

This is not a docs site. It is a practice repo. Each lesson is a folder with a real mini-codebase and a challenge you complete *inside Claude Code*. You learn by doing, not reading.

## How It Works

1. Open a lesson folder in your terminal
2. Read the `CHALLENGE.md` — it tells you what to do
3. Launch `claude` and complete the challenge
4. Check your work against the success criteria

## Prerequisites

- A computer (macOS, Windows, or Linux)
- An internet connection
- A Claude Pro, Max, Teams, or Enterprise account (or an API key)

No coding experience required for Lesson 01. Lessons 02+ assume basic terminal comfort.
