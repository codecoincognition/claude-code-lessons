# Lesson 02: Undo & Recover

**Time:** 20 minutes
**Level:** 2 — First Week
**Builds on:** Lesson 01 (install, @, !, /compact)

## What You'll Learn

1. **Esc×2** — the instant undo that rolls back Claude's last response
2. **/rewind** — selective rollback with control over how far back you go
3. **Code-only rollback** — undo the file changes but keep the conversation (Claude remembers what failed)
4. **The experimentation mindset** — when undo is free, experiments are free

## Why This Matters

Most people use Claude cautiously. Small prompts. Safe changes. Lots of hedging. They're afraid of making a mess.

This lesson teaches you that messes are free. Two keystrokes to undo. One command to go further back. And the best part: when you rewind, Claude remembers what went wrong — so the next attempt is smarter.

The developers and builders who get the most from Claude Code treat every prompt as an experiment. This lesson is where that mindset starts.
