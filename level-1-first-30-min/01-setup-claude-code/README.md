# Lesson 01: Your First 30 Minutes with Claude Code

**Time:** 25–30 minutes
**Level:** 1 — First 30 Minutes
**Prerequisites:** A computer, internet, and a Claude account (Pro, Max, Teams, or Enterprise)

## What You'll Learn

Seven things that make your first session feel magical — and set the right foundation for everything else:

1. **Install + first prompt** — Setup and the moment Claude writes code into your project
2. **`!` shell escape** — Run terminal commands without leaving Claude
3. **`@` file references** — Point Claude at exactly the files that matter
4. **`/compact`** — The one command that prevents 90% of frustration
5. **`/cost` + `/context`** — Know your token spend and context usage
6. **"Commands, not chat"** — The mindset shift that makes you 3x faster
7. **Context = Budget** — The mental model that changes how you use every AI tool

## Why This Matters

Most people open Claude Code, type a vague prompt, get a mediocre answer, and close the terminal. They never come back.

The developers who get 10x value from it learn these seven things in their first session. Not next week. Not after reading docs. In the first 30 minutes.

This lesson exists to make sure that's you.
