# Claude Code Micro Insights — Interactive Reference

A filterable, searchable reference card for the **Claude Code: Zero to Dangerous** curriculum.

**Live URL:** `https://codecoincognition.github.io/claude-code-lessons/insights.html`

---

## How to publish (GitHub Pages)

1. Drop `insights.html` into the root of this repo (or a `/docs` folder)
2. Go to repo **Settings → Pages**
3. Set source to `main` branch, `/ (root)` or `/docs`
4. GitHub will publish at:
   `https://codecoincognition.github.io/claude-code-lessons/insights.html`

No build step. No dependencies. Pure HTML.
